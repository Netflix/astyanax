astyanax by Netflix

Cassandra Java Client

Astyanax is a high level Java client for Apache Cassandra. Astyanax provides a connection pool as well as common best practices recipes on top of the cassandra thrift api.

Install

Get the binaries from Maven Central: 

<dependency>
    <groupId>com.netflix.astyanax</groupId>
    <artifactId>astyanax</artifactId>
    <version>version-number</version>
</dependency>
